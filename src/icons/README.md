## 生成
1. 进入ant-design-icons-master/packages/icons-svg/
2. 执行 generate 命令
3. 生成了包含名字空间的内联图标的 inline-namespaced-svg 文件夹
4. 拷贝 filled 和 outlined 里面的svg文件，twotone 风格不喜欢的就不要拷贝了
